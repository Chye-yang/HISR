# HISR - High-performance Invariant Sketch for Networking

HISR (高性能不变性网络测量框架) 是一个基于不变性学习的高性能网络流测量系统。

## 项目概述

HISR通过结合深度学习和传统Sketch技术，实现了在网络流量测量中的自适应和高精度性能。项目采用了多环境不变性学习策略，能够在不同网络条件下保持稳定的测量精度。

## 项目结构

```
HISR/
├── HISR开发指引/                     # 主要开发文档和代码
│   ├── HISRFigs/                    # 系统架构图
│   ├── 关键脚本实现示例/              # 核心算法实现
│   ├── 参考代码仓库/                 # 相关研究工作代码
│   ├── 主要参考工作/                 # 相关论文
│   └── 相关works/                    # 其他相关工作
├── README.md                        # 本文件
└── run_hisr_experiment.py          # 实验运行脚本
```

## 核心特性

- **多环境不变性学习**: 在不同网络环境下保持测量精度
- **自适应Sketch架构**: 动态调整测量策略
- **高性能实现**: 优化的数据处理管道
- **支持多种网络协议**: TCP、UDP等主流协议支持

## 快速开始

### 环境要求

- Python 3.8+
- PyTorch 1.8+
- 其他依赖见 `HISR开发指引/关键脚本实现示例（不一定完美复刻报告中的模型，仅参考）/requirements.txt`

### 安装依赖

```bash
pip install -r "HISR开发指引/关键脚本实现示例（不一定完美复刻报告中的模型，仅参考）/requirements.txt"
```

### 运行示例

```bash
python run_hisr_experiment.py
```
